🧠 1. 함수형 프로그래밍(Functional Programming)이란?

✔️ “함수를 가지고 수학처럼 프로그래밍하는 방식”

주요 특징은 아래 4개

✔️ (1) 함수를 값처럼 사용한다

즉,
	•	함수를 변수에 저장하고,
	•	다른 함수의 인자로 넘기고,
	•	함수에서 함수를 반환하고,

이런 게 자연스러움 → 고차 함수(Higher-Order Function)

✔️ (2) 데이터를 변경하지 않으려고 한다(불변성, immutability)

배열이나 객체를 “수정”하는 대신,
새로운 배열/객체를 만드는 방식을 선호함.

예) 원본 배열을 바꾸지 않고 map 사용
const arr = [1, 2, 3];
const arr2 = arr.map(x => x * 2);  // 새로운 배열 반환

✔️ (3) 순수 함수(pure function)를 사용한다

순수 함수란:
	•	같은 입력 → 항상 같은 출력
	•	함수 밖의 변수 변경 없음
	•	부작용(side effect)이 없음

const double = x => x * 2;  // 순수함수

✔️ (4) 함수 조합(function composition)을 선호한다

작은 함수들을 계속 조합해서 큰 기능을 만든다.

예)
const add = x => x + 1;
const double = x => x * 2;

const process = x => double(add(x));
console.log(process(3)); // 3 → 4 → 8
