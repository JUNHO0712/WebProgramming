나의 Web Programming 학습 홈페이지

- 제목 : HTML 문서 작성(실습)
- 이름 : 백준호
- 학부 : 컴퓨터공학과

- 2주차 학습 내용 : HTML 문서 구조와 기본 태그
학습 내용 구성 안내
  • 시맨틱 구조: <header>, <main>, <section>, <article>, <footer>
  • 접이식 소개: <details>, <summary>를 사용해 안내와 부가 설명을 토글
  • 주요 일정: 표(Table)로 주차별 주제를 요약
  • 상세 기록: 표 아래의 주차별 섹션에 과제/실습/메모를 기술 (필요 시 별도 HTML 파일로도 분리 가능)
학습 주차 강의 주제
1주강의 개요, 웹 개념/구조(HTTP, 클라이언트–서버), VS Code 설치 및 간단한 HTML 문서 작성
2주HTML5(교재 1–3장): 문서 구조, 기본 태그, 폼/멀티미디어, (실습) HTML 문서 작성
3주CSS(교재 4–5장): 문법과 스타일, 박스모델, 레이아웃 기법, (실습) CSS 적용
4주JavaScript 기초(교재 6장), 코어/사용자 객체(7장), (실습) JS 객체 만들기
5주DOM(8장): 요소 선택/조작, (실습) 동적 웹페이지 작성
6주이벤트(9장): 기본/핸들링, (실습) 이벤트 코드 작성
7주BOM(10장) & 캔버스 그래픽(11장), (실습) BOM 조작/그리기
8주중간고사
9주웹 스토리지(12장): 쿠키, 세션/로컬 스토리지, (실습) 스토리지 조작
10주오디오/비디오 제어 & 웹워커(13장), (실습) 미디어 컨트롤러/워커 통신
11주고급 JS: ES6, 데이터 타입, hoisting, 클로저/함수형 프로그래밍, (실습) Debugger

13주Express.js(1): 소개, 미들웨어, 라우터, (실습) 미들웨어 함수
14주Express.js(2): MongoDB 연동, CRUD API 작성
