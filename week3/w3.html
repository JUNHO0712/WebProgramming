학습 상세 내용

1주차: 웹 기본 개념과 환경 설정

1주차에는 웹의 기본적인 동작 원리에 대해 배웠습니다. 클라이언트가 서버에 HTTP 프로토콜을 통해 자원을 요청(request)하면, 서버는 그에 맞는 응답(response)을 보내주는 과정을 학습했습니다. 또한, 개발 환경 구축을 위해 Visual Studio를 설치하고 새로운 프로젝트를 생성하는 실습을 진행했습니다.

2주차: HTML 기본 구조와 태그

HTML이 웹 페이지의 뼈대를 구성하는 언어임을 배웠습니다. `

3주차: CSS 레이아웃 (Flexbox & Grid)

CSS를 통해 HTML 요소의 스타일을 지정하는 방법을 배웠습니다. 특히 레이아웃을 구성하는 현대적인 방식인 Flexbox와 Grid에 초점을 맞췄습니다. Flexbox는 1차원(한 방향) 레이아웃에, Grid는 2차원(행과 열) 레이아웃에 최적화되어 있음을 이해하고 각각의 기본 사용법을 실습했습니다.
